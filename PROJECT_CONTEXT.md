# Project Context

GrowthPilot AI is an AI-powered Lead Intelligence platform.

The MVP consists of one module:

Lead Intelligence.

The application performs deterministic analytics on uploaded historical lead datasets.

Artificial Intelligence is only used for interpretation and executive summaries.

The frontend computes all statistics.

The backend only proxies requests to NVIDIA.

No raw CSV data is ever sent to AI.

Future modules:

- Funnel Analytics
- Attribution
- Segmentation

Primary technologies:

- React
- TypeScript
- Vite
- Tailwind
- Recharts
- Netlify Functions
- NVIDIA NIM API

Always follow:

- Architecture.md
- UI_UX_GUIDELINES.md
- DATA_SPECIFICATION.md

Implement only the current phase.

Do not implement future functionality unless explicitly requested.
